---
name: commit
description: Analyze diff, write commit message
---

1. Run `git diff --staged` to see changes
2. Run `git diff` to see unstaged changes
3. Analyze what changed and why
4. Write a concise commit message following conventional commits:
   - feat: new feature
   - fix: bug fix
   - refactor: code restructure
   - docs: documentation
   - style: formatting
   - perf: performance
   - test: tests
5. Stage relevant files and commit
